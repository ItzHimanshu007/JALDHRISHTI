Technical Methodology

Hydrological modeling approach for rainfall-driven flood risk assessment in Jal Drishti

System Overview

Jal Drishti implements a raster-based water balance model that simulates pluvial (rainfall-driven) flooding. The system processes terrain data, applies hydrological algorithms, and classifies flood risk using multiple weighted criteria.

This system models:

Pluvial flooding (surface runoff from rainfall) in rural and semi-urban contexts across India

This system does NOT model:
  • River hydraulics or fluvial flooding
  • Dam/reservoir operations
  • Urban stormwater infrastructure
  • Groundwater interactions

Terrain Analysis (D8 Algorithm)

Slope Calculation

Slope is computed using the gradient of the DEM with a 3×3 moving window:

slope(degrees) = arctan(√(dz/dx² + dz/dy²)) × (180/π)

D8 Flow Direction

Flow Accumulation

Counts the number of upstream cells flowing into each cell. High flow accumulation indicates natural drainage channels where water concentrates during rainfall events.

Sink Filling:

Local depressions (sinks) in the DEM are filled using an iterative algorithm to ensure continuous flow paths.

Hydrological Water Balance Model

Core Equations

The water balance model calculates surface runoff based on rainfall input, infiltration, and terrain characteristics:

Infiltration: I = C_land × R × (1 - saturation)
Where C_land = infiltration coefficient, R = rainfall, saturation = soil saturation level
Surface Runoff: Q = R - I
Where Q = surface runoff (mm), R = rainfall (mm), I = infiltration (mm)
Water Depth: D = (Q × A_upstream) / A_cell
Slope constraint applied: water depth reduced on slopes > 15°

A gradient boosting model provides seasonal risk assessment using terrain, climate, and historical flood features.

  1. Feature Extraction: Terrain stats, rainfall patterns, building density, historical floods
  2. Seasonal Modeling: Monsoon multiplier applied (2.0× during June–September)
  3. Calendar Generation: Monthly risk predictions for yearly planning

Limitations & Future Work

Current Limitations

  • Does not account for existing drainage infrastructure
  • Assumes uniform rainfall distribution (no spatial variation)
  • Simplified infiltration model (no detailed soil data)
  • Static land cover (no seasonal variation)

Recommended Enhancements

  • Integration with IMD real-time rainfall forecasts
  • Higher resolution DEM (5–10m) for urban areas
  • Ground validation with historical flood extent data
  • Dynamic hydraulic routing for large catchments
